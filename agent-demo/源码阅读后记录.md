### 核心启动链路

```
bootstrap-entry.ts          → 最小引导：注入全局宏
  └─ bootstrapMacro.ts      → 设置 VERSION/PACKAGE_URL 等
  └─ entrypoints/cli.tsx    → CLI 入口：Commander 参数解析
       └─ main.tsx          → 主编排：初始化 → REPL 启动
            ├─ init.ts      → 运行时初始化（遥测/策略/MCP）
            ├─ REPL.tsx     → 交互式主屏
            └─ QueryEngine  → 会话查询引擎
                 └─ query.ts → 单次请求管道（消息组装/工具调用/重试）
```

```cli.tsx main() 路由分发：
--version / -v → 直接打印版本号，return（零模块加载的快速路径）
--dump-system-prompt → 输出系统提示，return
--claude-in-chrome-mcp → Chrome MCP 服务
--daemon-worker → 守护进程 worker
remote-control / bridge → 远程桥接
daemon → 守护进程
ps / logs / attach / kill → 后台会话管理
new / list / reply → 模板任务
--worktree --tmux → tmux 工作树

（以上全是 fast path，提前 return）

默认路径 → await import('../main.js') → cliMain()
          → 启动完整 REPL 交互界面
```
