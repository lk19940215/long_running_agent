# 文件上传

## 任务分解指导
文件上传通常嵌入弹窗表单中，不单独拆任务。如上传逻辑较复杂（多文件、进度条、预览），可拆为独立 frontend 任务。

## 实现要点
- 文件类型限制：accept 属性 + 前端校验
- 文件大小限制：前端校验 + 后端校验
- 上传进度：progress bar（XMLHttpRequest 或 fetch + ReadableStream）
- 上传成功后：显示文件名/预览图 + 返回 fileId
- 多文件上传：列表展示已上传文件，支持删除

## 验证策略
- 选择超限文件 → 验证拒绝提示
- 上传正确文件 → 验证成功提示 + 预览显示
